# Gestion automatique des dépenses personnelles

Ce projet vise à mettre en place un pipeline **ETL minimal en Python** pour automatiser la collecte, la transformation et la sauvegarde des dépenses personnelles.

> **Statut : en cours de développement** — les fonctionnalités évoluent progressivement.

## Exécution

```bash
python main.py
```

Les fichiers sources doivent être placés dans `data/raw/`.
